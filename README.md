**Delivery:** Online LMS + Video lectures + Interactive workshops

## CURRICULUM STRUCTURE - 5 MODULE PROGRESSION

### MODULE 1: Classifying and Sourcing Data (Weeks 1-2)
- **Duration:** 4 hours instruction + 8 hours self-study
- **Learning Hours:** 12 hours total
- **Key Topics:**
  * What is Data?
  * Data Analysis vs Data Analytics
  * Primary vs Secondary Data
  * Quantitative vs Qualitative Data
  * Structured vs Unstructured Data
  * Data Inclusion & Exclusion Criteria
  * Real-World Applications & Case Studies

**Learning Outcomes:**
✓ Define data and understand its role in decision-making
✓ Differentiate between primary and secondary data sources
✓ Distinguish quantitative data from qualitative data
✓ Contrast structured data with unstructured data
✓ Evaluate data quality and validity

---

### MODULE 2: Data Cleaning and Preparation (Weeks 2-4)
- **Duration:** 5 hours instruction + 10 hours hands-on practice
- **Learning Hours:** 15 hours total
- **Key Topics:**
  * Data Quality Dimensions
  * Data Cleaning Techniques
  * Working with Excel
  * Data Validation & Verification
  * Data Documentation

**Learning Outcomes:**
✓ Identify and document data quality issues
✓ Handle missing, duplicate, and inconsistent data
✓ Standardize and format data for analysis
✓ Validate data integrity and completeness
✓ Create data documentation and metadata

---

### MODULE 3: Data Visualization (Weeks 5-7)
- **Duration:** 6 hours instruction + 12 hours projects
- **Learning Hours:** 18 hours total
- **Key Topics:**
  * Visualization Principles
  * Chart Types & When to Use Them
  * Introduction to Tableau/Power BI
  * Color, Design & Accessibility
  * Data Storytelling Techniques

**Learning Outcomes:**
✓ Apply data visualization best practices
✓ Create effective charts and graphs
✓ Design visualizations for different audiences
✓ Use visualization tools effectively
✓ Tell compelling stories with data

---

### MODULE 4: Statistical Analysis (Weeks 8-11)
- **Duration:** 7 hours instruction + 14 hours practice
- **Learning Hours:** 21 hours total
- **Key Topics:**
  * Descriptive Statistics
  * Probability Distributions
  * Hypothesis Testing
  * Correlation & Regression
  * Statistical Software (R or Python basics)
  * Research Design & Sampling

**Learning Outcomes:**
✓ Understand statistical concepts and methods
✓ Perform descriptive statistical analysis
✓ Conduct hypothesis testing
✓ Interpret statistical results correctly
✓ Avoid common statistical pitfalls

---

### MODULE 5: Reporting and Dashboards (Weeks 12-15)
- **Duration:** 8 hours instruction + 16 hours capstone project
- **Learning Hours:** 24 hours total
- **Key Topics:**
  * Report Writing & Structure
  * Dashboard Design Principles
  * Advanced Tableau/Power BI
  * Presentation Skills
  * Documentation & Knowledge Sharing
  * Capstone Project - Full Data Pipeline

**Learning Outcomes:**
✓ Design professional data reports
✓ Build interactive dashboards
✓ Communicate findings to stakeholders
✓ Present insights and recommendations
✓ Create end-to-end data solutions
